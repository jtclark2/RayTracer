# Usage
Run the executable, redirecting the output into an image.ppm file:
```
> cd bin\Win32\Debug
> RayTracing.exe > image.ppm
```
To open ppm files, consider using:
- Gimp
- [This Online Viewer] (https://www.cs.rhodes.edu/welshc/COMP141_F16/ppmReader.html)
